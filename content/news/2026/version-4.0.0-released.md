Version 4.0.0 was released today, which includes security fixes and new features.

As a breaking change, the Corona tracing app is removed.
Furthermore, non-guessable validation URLs are enforced now, which were introduced in version 3.0.0.

Details can be found in the [changelog](https://docs.helfertool.org/releases/changelog.html#changelog-4-0-0).
